# CS 463/563 – Intro to Web Development Final Project

## My Portfolio

This is a simple portfolio website about me, my professional background, projects and contact me forum built using HTML, CSS, and JavaScript.
It includes the following pages:

- Home
- About Me
- Previous Work
- Projects
- Contact

## Getting Started

- To run the website, simply open the index.html file and run it in your web browser.

## Features

- Navigation bar: Allows users to easily navigate between different pages.
- Dynamic content loading: Uses JavaScript to fetch and display HTML content without refreshing the page.
- Contact form: Validates user input and displays an alert message when the form is submitted successfully.

## Built With

- HTML
- CSS
- JavaScript
- Font Awesome (for icons)
